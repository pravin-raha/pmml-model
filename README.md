# pmml-model

## Requirements

For development, you will only need Java and SBT installed on your environment.

## Build

    $ git clone https://github.com/pravin-raha/pmml-model.git
    $ cd pmml-model
    $ sbt clean assembly
    
## How to Run

    $ java -jar pmml-model-assembly-0.1.0-SNAPSHOT.jar $PORT
